# nesdump2ff

Dumps the character rom to a farbfeld image to the stdout

# Installation

```
$ git clone https://github.com/GabrielMtins/nesdump2ff.git
$ cd nesdump2ff
$ make
$ sudo make install
```

# Dependencies

libgffimage: https://github.com/GabrielMtins/libgffimage

# Usage

```
$ nesdump2ff file.nes > file.ff
```
